# KASACLE - 筋トレ管理アプリ

筋トレのワークアウトを記録・管理するiOSアプリです。セット数・レップ数・重量・時間を記録し、カレンダーでトレーニング履歴を振り返ることができます。

## 機能

### ワークアウト記録
- 6つの部位（胸・背中・肩・腕・脚・腹）からエクササイズを選択
- セットごとにレップ数・重量（自重対応）・時間を記録
- インターバルタイマー付き
- トレーニング終了時にサマリーを表示

### カレンダー
- 月別のトレーニング履歴をカレンダー上に表示
- 日付をタップしてその日のワークアウト詳細を確認

### 今日のアドバイス
- 曜日に応じたトレーニングアドバイスを表示

### エクササイズ設定
- 部位ごとのエクササイズ一覧を管理
- カスタムエクササイズの追加・削除

## 技術スタック

- **言語**: Swift
- **UI**: SwiftUI
- **データ永続化**: SwiftData
- **対応OS**: iOS 17+
- **外部ライブラリ**: なし（Apple純正フレームワークのみ）

## プロジェクト構成

```
kasacle-app/kasacle-app/
├── App/            # アプリエントリーポイント、ルートビュー
├── Views/
│   ├── Home/       # ホーム画面、今日のアドバイス
│   ├── Workout/    # ワークアウト記録、カレンダー、日別詳細
│   ├── Settings/   # エクササイズ管理
│   └── Splash/     # スプラッシュ画面
├── Models/         # データモデル（WorkoutRecord, WorkoutSession, WorkoutSet 等）
└── Theme/          # カラーパレット
```

## ビルド方法

1. Xcode 16以降で `kasacle-app.xcodeproj` を開く
2. シミュレータまたは実機を選択してビルド・実行
